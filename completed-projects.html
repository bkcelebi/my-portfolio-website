Project: buddydrivers

Project Overview

The project was to create an easily navigable environment for prospective driving test candidates, who do not have the facilities or opportunity to practise outside EDT lessons, to find an experienced driver in their local area and/or speaks their preferred language.

It allows learners and experienced drivers across Ireland to connect and communicate by providing necessary contact information and a messaging feature, which allows users to access both sent and received messages.

It also provides experienced drivers with the opportunity to exploit their driving expertise, maximise their free time with the potential scope of becoming an Approved Driving Instructor (ADI), and, if desired and feasible, the possibility of increasing their monthly income.

PS: This project is not being hosted at the moment

Project: IBM Watson Sentiment Analyser

Project Overview

A simple online sentiment analyser built with React. The project uses IBM Watson Sentiment Analyser to analyse the sentiment and/or emotion from the user input, and is hosted on the IBM Cloud as a Cloud Foundry application. The user can submit text or provide a URL for analysis, and choose between sentiment or emotion analysis. The input has to be more than ~30 characters in order for IBM Watson to be able to analyse it.

Project: Movie Reviews Database

Project Overview

This project utilizes the Natural Language Processing (NLP) capabilities of the IBM Cloud API to analyze movie reviews, and IBM Cloud pipeline to deploy the code. The API is used to determine the sentiment of a given review, as well as identify key entities and concepts mentioned in the review.

Using the IBM Cloud NLU service and IBM Cloud pipeline, it is possible to quickly and easily analyze movie reviews to determine sentiment and identify key entities and concepts. This can provide valuable insights for movie studios and reviewers. With pipeline, the deployment process is automated and streamlined, making it easy to manage different environments, and rollback if necessary.

PS: This project was done as a part of Full Stack Software Developer Professional Certificate however, it is not being hosted at the moment
